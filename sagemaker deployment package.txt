Let's see how to create sagemaker deployment package for sagemaker.
sagemaker isn't installed in lambda functions
Let's see the steps on how to do that

1- open jupyter lab: Open a jupyter notebook ()
2- Open terminal : file > New > Terminal >
3- Check python version in terminal: python3 --version
4- Checl if sagemaker is installed: 
 	-python3 to open python
	-import sagemaker to see if sagemaker is installed
	- exit() 
5- Install sagemaker in terminal: 
	- check your current directory path
	- cd to your current directory
	- create a `python` folder in your current directory
	- cd to the `python` folder
	- pip install sagemaker -t python/


6- create python file named 'lambda_function' in `python` directory:
	- touch lambda_function.py
7- Put content to this pyton file
	- nano lambda_function.py (opens it)
	import json
	import sagemaker

8- Install zip package: 
using:
	sudo apt update
	sudo apt install zip -y
or using:
	conda install zip -y
9- zip the content of 'python' directory : Dot for all 
	`zip -r sagemaker_deployment_package.zip .`

10- Upload the zipped file to s3:
	import boto3
